# OLIMPIADAS-TEORIA-ORGANIZACIONAL

Aplicación interactiva gamificada diseñada para fortalecer el aprendizaje en Teoría Organizacional mediante retos dinámicos por equipos.

Esta experiencia educativa integra conceptos clave como:
- Tipología de empresas  
- Estructura organizacional  
- Necesidades de información  
- Sistemas contables  

A través de un sistema de preguntas, retroalimentación inmediata y puntuación acumulativa, los estudiantes desarrollan pensamiento crítico, toma de decisiones y comprensión aplicada de los contenidos.

## Características principales
- Juego por equipos con marcador en tiempo real  
- Retos por módulos temáticos  
- Sistema de puntuación y ranking final (podio)  
- Temporizador por pregunta  
- Retroalimentación explicativa  
- Modo oscuro y claro  
- Diseño optimizado para dispositivos móviles  

## Objetivo
Transformar el aprendizaje tradicional en una experiencia activa, competitiva y significativa dentro del aula universitaria.
